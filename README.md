# Aircraft War Game

This is a 2D Aircraft War Game built using Python and Pygame. The player controls an aircraft, shoots bullets at opponents, and avoids getting hit.

## Features
- Pygame-based rendering and audio
- Multiple enemies with collision detection
- Scoring system and game over logic

## Installation
- Install Pygame: `pip install pygame`
- Run the game: `python main.py`

## Assets
All game assets (images, sounds) are included in the `image/` folder.
